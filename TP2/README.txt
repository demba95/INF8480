•	Placez-vous dans le dossier du Projet et lancez la commande ant suivante pour compiler le projet et deplacez vous dans le dossier bin

•	Ouvrez un terminal dans le dossier du projet et lancez le serveur LDAP avec la commande ./ldap.sh

•	Ouvrez un autre terminal et lancez le serveur avec la commande
			./server.sh [numero port] [capaciteServeur m] [taux de Malice q]
•	Lancez ensuite le Répartiteur avec la commande (Mode = s pour sécurisée et n pour non sécurisée)
./client.sh [fichier] [mode]
